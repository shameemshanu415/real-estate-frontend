# 🏠 Real Estate Price Prediction System - Frontend

This is the frontend (UI) of the Real Estate Price Prediction System.  
It is built using React and communicates with the Django backend API.

---

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Property Price Prediction Form
- AI Chatbot (Gemini AI Integration)
- Analytics Dashboard
- Modern UI using ShadCN + Tailwind CSS

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- ShadCN UI Components
- Axios / Fetch API
